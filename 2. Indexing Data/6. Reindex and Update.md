# Use the Reindex API and Update By Query API to reindex and/or update documents

## Reindex
[Reindex API documentation.](https://www.elastic.co/guide/en/elasticsearch/reference/7.2/docs-reindex.html)

Example:
```json
POST _reindex
{
  "source": {
    "index": "[original_index]"
  },
  "dest": {
    "index": "[new_index]"
  }
}
```
